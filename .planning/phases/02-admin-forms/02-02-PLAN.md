    **Add PATCH to `src/app/api/admin/vehicles/[id]/images/route.ts`:**

    Add new PATCH export function. Handles two operations:

    1. **Reorder**: If body contains `images` array: iterate `body.images`, for each `{ id: imageId, sortOrder }`, update `vehicle_images` row where `id = imageId AND vehicle_id = vehicleId` set `sort_order = sortOrder`.

    2. **Primary toggle**: If body contains `primaryImageId`: first unset all `is_primary = false` for `vehicle_id`, then set `is_primary = true` for the specified image.

    Full implementation:
    ```typescript
    export async function PATCH(req: NextRequest, { params }: RouteParams) {
      try {
        const { id } = await params
        const body = await req.json()
        const { supabase, error: authError } = await requireAuth()
        if (authError) return authError

        if (body.images && Array.isArray(body.images)) {
          for (const img of body.images) {
            await supabase
              .from("vehicle_images")
              .update({ sort_order: img.sortOrder })
              .eq("id", img.id)
              .eq("vehicle_id", id)
          }
        }

        if (body.primaryImageId) {
          await supabase
            .from("vehicle_images")
            .update({ is_primary: false })
            .eq("vehicle_id", id)
          await supabase
            .from("vehicle_images")
            .update({ is_primary: true })
            .eq("id", body.primaryImageId)
            .eq("vehicle_id", id)
        }

        return NextResponse.json({ success: true })
      } catch {
        return NextResponse.json({ error: "Failed to update images" }, { status: 500 })
      }
    }
    ```

    Keep existing POST and DELETE handlers unchanged.
