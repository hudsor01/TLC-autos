4. Create `src/app/api/inventory/route.ts` -- public GET endpoint (no auth required):
   - Import createClient from `@/lib/supabase/server`, sanitizeSearch from `@/lib/utils`
   - Define `ALLOWED_SORT = ["date_added", "selling_price", "mileage", "year"]`
   - Define a SORT_MAP to translate client sort names to DB columns: `{ newest: "date_added", "price-low": "selling_price", "price-high": "selling_price", mileage: "mileage", year: "year" }`
   - Define ORDER_MAP for ascending/descending: `{ newest: false, "price-low": true, "price-high": false, mileage: true, year: false }`
   - Parse searchParams: search, make, model, yearMin, yearMax, priceMin, priceMax, sort, page
   - Default limit = 12 (grid-friendly for 3-4 columns)
   - Build Supabase query on "vehicles" table with select `*, vehicle_images(url, sort_order)` and `{ count: "exact" }`
   - Always filter `.eq("status", "available")` (public only sees available)
   - Conditionally apply: `.eq("make", make)` if make, `.eq("model", model)` if model, `.gte("year", yearMin)` if yearMin, `.lte("year", yearMax)` if yearMax, `.gte("selling_price", priceMin)` if priceMin, `.lte("selling_price", priceMax)` if priceMax
   - If search: `.or(\`make.ilike.%${sanitizeSearch(search)}%,model.ilike.%${sanitizeSearch(search)}%\`)`
   - Resolve sort column via SORT_MAP (default "date_added"), validate against ALLOWED_SORT
   - Resolve ascending via ORDER_MAP (default false)
   - Apply `.order(validSort, { ascending }).range(from, from + limit - 1)`
   - Transform results: for each vehicle, sort vehicle_images by sort_order, map to URL strings, convert snake_case vehicle fields to camelCase using camelKeys (but handle vehicle_images separately since it's a join)
   - Also query for filter options in the same handler: run a separate query `supabase.from("vehicles").select("make, model, year, selling_price").eq("status", "available")` to compute distinct makes, models (optionally filtered by selected make), year range, price range
   - Return JSON: `{ vehicles: [...], pagination: { page, limit, total, totalPages }, filterOptions: { makes: string[], models: string[], yearRange: { min, max }, priceRange: { min, max } } }`
   - Wrap in try/catch, return 500 on error
